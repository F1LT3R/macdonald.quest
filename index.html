What are Triggers?

Triggers are custom commands we are programming at home to give us super-powers. We are building a library of triggers to create team games and competitions in Minecraft. If you want to use the triggers too, you will need to request an /op from DaddosaurusMacs. Trigger scripts are available on GitHub: minecraft-datapacks.
